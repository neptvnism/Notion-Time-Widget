The Notion Clock Widget is a customizable clock widget designed to be embedded in Notion pages. 

## Features
- **Dark Mode**: Toggle between light and dark themes. The best part is, it has the same shade of dark as notion! This will blend perfectly in your workspace :)
- **Shadows**: Enable or disable text shadows for the clock.
- **Date Display**: Option to show or hide the current date.
- **Font Selection**: Choose from different fonts for the clock display, the same fonts offered by notion too!
- **12-Hour/24-Hour Format**: Select between 12-hour and 24-hour time formats.

Enjoy!
